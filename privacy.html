Privacy Policy

Last updated: March 18, 2026

MailMapr does not collect, transmit, or share any personal data. All email analysis is performed entirely on your Mac. No data ever leaves your device.

1. Data Collection

MailMapr does not collect any personal information. Specifically:

  • No user accounts or registration required
  • No analytics or telemetry data collected
  • No usage tracking or behavioral profiling
  • No crash reports sent to external servers
  • No advertising identifiers used
  • No cookies or web tracking

2. Email Data Processing

MailMapr processes email files (EML, EMLX, MSG, MBOX) that you explicitly import into the application. This processing:

  • Occurs entirely on your local device
  • Is stored only in the application's sandboxed container on your Mac
  • Is never transmitted to any external server or cloud service
  • Can be deleted at any time by removing projects within the app

3. Network Usage

MailMapr includes a network entitlement solely to support report generation features (HTML rendering). The application:

  • Does not make any outbound network requests to external servers
  • Does not phone home or check for updates externally
  • Does not transmit any user data over the network

4. Data Storage

All application data is stored locally within the macOS Application Support directory, protected by the App Sandbox. This includes:

  • Imported email artifacts and their analysis results
  • Project configurations and trust lists
  • Generated forensic reports
  • Application preferences

You have full control over this data and can delete it at any time.

5. Third-Party Services

MailMapr does not integrate with any third-party services, SDKs, or analytics platforms. There are no external dependencies that process user data.

6. Children's Privacy

MailMapr is a professional forensic tool and does not target or collect information from children under 13.

7. Changes to This Policy

If we update this privacy policy, the changes will be posted on this page with an updated revision date.

8. Contact

If you have questions about this privacy policy, please contact us at:
